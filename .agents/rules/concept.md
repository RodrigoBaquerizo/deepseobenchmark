El objetivo es crear una aplicación que sirva para automatizar el proceso de benchmark de un agente SEO.

El usuario introducirá los dominios o URLs que desea examinar y el agente obtendrá información de la web (título, metadescripción, schema markup, etc.) e interpretará el contenido y la arquitectura de la web a traves de Gemini, para entender conceptos y objetivos de cada web examinada y así obtener mejores insights.
